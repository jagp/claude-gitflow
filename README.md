# claude-gitflow
Claude doesn't play well with gitflow patterns out of the box;  what starts as a bit of a nuisance, quikly spoirals into a messy, obfuscated, coumnterintuitive slopfest, breaking the core advantages of version control.  

This package of atomic, composable tools help Claude and his agentic army of agents "go with the flow".  The core is a package of "set it and forget it" hooks that ensure compliance with GitFlow patterns - codified guardrails, not vague suggestions:

Every hook will just work without ijntervention, but if your team's setup requires nuamnce, everything comes in a coinfigurable/overrideable format.
